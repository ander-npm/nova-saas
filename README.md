# Nova – Minimal SaaS Landing Template

Modern, minimalist React + Tailwind CSS landing template built for developers who want to launch fast.

---

## Why Nova?

Nova is built with a clean architecture and scalable structure so you can:

- Launch your SaaS in minutes
- Skip repetitive setup
- Focus on product, not boilerplate
- Use a professional UI foundation


## Features

- Vite + React + TypeScript
- TailwindCSS v4
- Dark Mode (default) + Light Mode
- Smooth animations with Framer Motion
- Clean folder architecture
- Reusable UI components
- Design tokens centralized
- ESLint configured
- Fully responsive
---

## Tech Stack

- React

- TailwindCSS
- Framer Motion
- Vite

## Getting Started

1. Install dependencies

```bash
npm install
```

2. Run development server

```bash
npm run dev
```

3. Build for production

```bash
npm run build
```

---

## Customization

### Change colors

Edit design tokens in:

```
src/index.css
```

### Modify pricing

Edit:

```
src/data/pricing.ts
```

### Edit sections

Go to:

```
src/components/sections/
```
